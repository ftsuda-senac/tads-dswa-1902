Exemplo Chamada AJAX

NOTAR QUE esta tela não usa nenhum recurso do Spring MVC ou Thymeleaf

  • Ver a classe ProdutoRestController e comparar a lógica e anotações com ProdutoController
    • O ProdutoRestController foi configurado com @RestController; por isso, o retorno por padrão será um JSON dos objetos retornados por cada método.
    • Abrir as URLs configuradas como @GetMapping emProdutoRestController para ver o JSON retornado
    • Alguns atributos das classes de Entidade foram anotadas com @JsonIgnore para evitar referência circular
  • Ver o conteúdo do arquivo ajax.js que contém uma chamada AJAX para o ProdutoRestController para obter a lista dos produtos e mostrar dinamicamente as informações na tela.
